# Detecting Web Attacks - TryHackMe Walkthrough

This repository contains my walkthrough for the Detecting Web Attacks room on TryHackMe.

The room focuses on identifying common web attacks through log analysis and packet inspection. It includes detecting directory fuzzing, brute-force login attempts, SQL injection activity, and reviewing HTTP traffic in Wireshark. It also touches on creating simple WAF rules to block suspicious requests.

The full walkthrough is included as a PDF file.
